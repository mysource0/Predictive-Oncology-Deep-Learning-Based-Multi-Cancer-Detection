🏗️ Model Architecture

Convolutional Neural Network (CNN)

The current model uses a custom CNN architecture designed for medical image classification:

Conv2D(32, 3x3) + ReLU MaxPooling2D Conv2D(64, 3x3) + ReLU MaxPooling2D Flatten Dense(128) + ReLU Dense(N_classes) + Softmax

Model Specifications

  • Input Size: 150×150×3 (RGB images)
  • Optimizer: Adam
  • Loss Function: Sparse Categorical Crossentropy
  • Metric: Accuracy
  • Training Epochs: 10
  • Validation Split: 20%

🧬 Predictive Oncology

Deep Learning-Based Multi-Cancer Detection System

📌 Introduction

Predictive Oncology focuses on using artificial intelligence and deep learning techniques to detect cancer at an early stage. Early detection significantly improves treatment success and survival rates.

This system analyzes medical images using advanced deep learning models to classify multiple cancer types including Breast, Liver, and Skin cancers. It helps doctors and researchers make faster and more accurate decisions.

🎯 Objectives

  • Detect multiple cancer types using deep learning
  • Classify images into normal, benign, and malignant categories
  • Assist early diagnosis and reduce medical errors
  • Provide automated and fast predictions
  • Support healthcare professionals in decision making

🧠 Deep Learning Models

CNN Custom Model

Convolutional Neural Networks automatically extract important features such as texture, shape, and patterns from medical images. It is widely used for medical image classification.

VGG16 Transfer Learning

VGG16 is a pre-trained deep learning model that helps extract high-level features efficiently. It improves accuracy even with limited medical datasets.

ResNet50 Deep Architecture

ResNet50 uses residual connections to allow very deep networks. It avoids vanishing gradient problems and performs well on complex medical data.

⚙️ System Workflow

  • Upload medical image (Breast, Liver, or Skin)
  • Resize image to 150×150 pixels
  • Normalize pixel values
  • Extract features using deep learning models
  • Classify into cancer type
  • Display prediction with medical suggestion
